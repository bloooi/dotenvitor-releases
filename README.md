# dotenvitor-releases

[envitor](https://github.com/bloooi/envitor) 앱의 릴리스 바이너리(`.dmg`)와 [Sparkle](https://sparkle-project.org/) 업데이트 피드(`appcast.xml`)를 호스팅합니다.

## 업데이트 피드

- **Appcast URL**: <https://bloooi.github.io/dotenvitor-releases/appcast.xml>
- **다운로드**: [Releases 페이지](https://github.com/bloooi/dotenvitor-releases/releases)

앱 안의 "envitor → 업데이트 확인…" 메뉴가 위 피드를 주기적으로 조회합니다.

## 개발자 참고

릴리스 파이프라인과 앱 소스 코드는 [bloooi/envitor](https://github.com/bloooi/envitor) 에 있습니다.
