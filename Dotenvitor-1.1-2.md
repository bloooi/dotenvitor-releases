# Dotenvitor 1.1

## 새로운 기능

### Skill 한번에 설치

이제 Dotenvitor 안에서 **Skills 설치** 버튼 한 번으로
사용 중인 AI 코딩 도구(Claude Code, Codex, Gemini 등)에
Dotenvitor skills 을 설치할 수 있습니다.

설치된 skills 는 AI 가 `.env` 파일을 직접 편집하지 않도록 안내해,
시크릿이 AI 컨텍스트에 노출되는 위험을 줄여 줍니다.

## 안내

이 버전부터 앱 안정성 향상을 위해 익명 사용 통계와 크래시 보고가
수집됩니다. **앱 설정 → 개인정보** 에서 끌 수 있습니다.

[자세한 내용](https://bloooi.github.io/dotenvitor-releases/privacy.html)
