Dotenvitor 개인정보 처리방침

시행일: 2026-05-05

※ 본 방침은 Dotenvitor 1.1 이상에 적용됩니다. 이전 버전(1.0)은 어떠한 사용 통계나 크래시 데이터도 수집하지 않습니다.

본 개인정보 처리방침은 Dotenvitor (이하 "본 앱")이 사용자의 개인정보를 어떻게 처리하는지 설명합니다.

1. 처리 목적

본 앱은 다음 목적을 위해 개인정보를 수집·이용합니다.

  • 앱 안정성 향상 (크래시 분석 및 디버깅)
  • 익명 사용 통계를 통한 기능 개선

2. 처리 항목

본 앱이 수집하는 항목은 다음과 같습니다.

(가) 자동 수집 (옵트아웃 가능)

  • 익명 식별자 (UUID, 앱 설치 시 무작위 생성)
  • 운영체제 버전, 앱 버전, 빌드 번호
  • 시스템 로케일 및 선호 언어
  • 앱 사용 이벤트 (예: 프로젝트 추가, 파일 편집 모드 전환 등의 카운트)
  • 크래시 발생 시 스택 추적 및 디바이스 모델 정보

(나) 본 앱이 절대 수집하지 않는 정보

  • 사용자가 편집하는 .env 파일의 키 이름, 값, 내용
  • 파일 경로, 파일명, 프로젝트 디렉터리명
  • IP 주소
  • 사용자가 입력한 자유 텍스트
  • 이메일, 이름, 전화번호 등 개인 식별 정보

3. 보유 기간

  • Sentry (크래시 보고): 90일
  • PostHog (사용 통계): 7년 (각 서비스 기본값)
  • 사용자가 옵트아웃하면 즉시 더 이상의 데이터를 전송하지 않습니다.

4. 제3자 위탁

본 앱은 다음 외부 서비스를 데이터 처리자로 이용합니다.

위탁받는 자 위탁 업무 내용 소재지
Functional Software, Inc. (Sentry) 크래시 보고 수집 및 분석 미국
PostHog Inc. 익명 사용 통계 수집 및 분석 미국

5. 개인정보의 국외 이전

위 위탁에 따라 개인정보가 미국으로 이전됩니다. 이전되는 항목은 "2. 처리 항목"의 자동 수집 항목과 동일합니다.

6. 정보주체의 권리

사용자는 다음 권리를 행사할 수 있습니다.

(가) 옵트아웃 (수집 중단)

앱 메뉴 > 설정 > 개인정보 > "익명 사용 통계 공유" 토글을 끄면 즉시 데이터 전송이 중단됩니다.

(나) 데이터 삭제 요청

이메일로 본인의 익명 식별자(UUID)와 함께 삭제를 요청하시면 관련 외부 서비스에서 해당 데이터를 삭제하도록 조치합니다.

8. 처리방침의 변경

본 처리방침은 관련 법령, 정책 또는 보안 기술의 변경에 따라 개정될 수 있습니다. 변경 시 본 페이지에 공지합니다.
