Dotenvitor

Gather your project's .env files in one focused editor, and keep sensitive values under your control.

Download for Mac Requires macOS 15.0 or later
01

Open a project once

Choose a project folder and keep its environment files ready for quick edits.

02

Find every .env file

Root and nested environment files appear as tabs, so switching contexts stays simple.

03

Edit as form or text

Use clean fields for routine changes, or switch to raw text when the file layout matters.

04

Works with dotenvx

Dotenvx keys and encrypted values stay visible, clear, and separate from ordinary entries.

05

Duplicate env files

Copy a file with all values intact, or create a keys-only version for a fresh environment.

06

Group with comments

Turn descriptive comments into clear sections that keep related keys together.

07

Keep notes in view

Inline comments remain attached to each row, so the context stays visible while you edit.

08

Restore commented keys

See disabled keys in the editor and bring them back when the configuration needs them.

09

Reveal in Finder

Open the selected project, folder, or environment file directly in Finder.

10

Catch duplicates early

Repeated keys and values are highlighted where you edit, before they become easy to miss.
